# ItemPhysic Guns

**[EN]**
This client-side mod acts as a bridge between **ItemPhysic** (Full or Lite) and popular 3D gun mods. It applies custom transformations to dropped weapon items to ensure they lie flat and look natural, instead of clipping through the ground or floating awkwardly.

### Key Features
- **Smart Transformations**: Corrects the rendering of dropped weapons from all supported mods.
- **Universal**: Works with both **ItemPhysic Full** and **ItemPhysic Lite**.
- **Lightweight & Client-Side**: Has no impact on server performance.
- **Extensible**: Other mod developers can easily add support for their own items.

### Supported Mods
This addon provides out-of-the-box support for the following mods, as well as any others whose developers have integrated compatibility using the provider system.

- **Required (at least one):**
    - **ItemPhysic Full** (Tested with version `1.8.9`)
    - **ItemPhysic Lite** (Tested with version `1.6.6`)

- **Guns (Optional):**
    - **Timeless and Classics Zero (TaCZ)** (Tested with version `1.1.6`)
    - **Superb Warfare** (Tested with version `0.8.8`)
    - **Vic's Point Blank** (Tested with version `1.9.6`)
    - **Scorched Guns 2** (Tested with version `0.5.5`)

### For Mod Developers
If you want your mod's 3D items to be supported, you can easily add compatibility without requiring this mod as a dependency.

**[RU]**
Этот клиентский мод служит связующим звеном между **ItemPhysic** (обычным или Lite) и популярными модами на 3D-оружие. Он автоматически корректирует положение и вращение выброшенных предметов, чтобы они естественно лежали на поверхности, а не проваливались под текстуры или странно парили в воздухе.

### Ключевые особенности
- **Умные трансформации**: Исправляет отображение лежащего оружия из всех поддерживаемых модов.
- **Универсальность**: Работает как с **ItemPhysic Full**, так и с **ItemPhysic Lite**.
- **Лёгкость и работа на клиенте**: Мод не влияет на производительность сервера.
- **Расширяемость**: Разработчики других модов могут легко добавить поддержку для своих предметов.

### Поддерживаемые моды
Аддон обеспечивает встроенную поддержку для следующих модов, а также для любых других, чьи разработчики добавили совместимость через систему провайдеров.

- **Обязательные (хотя бы один):**
    - **ItemPhysic Full** (Протестировано с версией `1.8.9`)
    - **ItemPhysic Lite** (Протестировано с версией `1.6.6`)

- **Оружейные (опционально):**
    - **Timeless and Classics Zero (TaCZ)** (Протестировано с `1.1.6`)
    - **Superb Warfare** (Протестировано с `0.8.8`)
    - **Vic's Point Blank** (Протестировано с `1.9.6`)
    - **Scorched Guns 2** (Протестировано с `0.5.5`)

### Для разработчиков модов
Если вы хотите, чтобы 3D-предметы из вашего мода поддерживались, вы можете легко добавить совместимость, не требуя этот мод в качестве зависимости.
